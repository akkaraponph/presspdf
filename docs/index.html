Getting Started

Hello World

package main

import "github.com/akkaraponph/folio"

func main() {
    doc := folio.New()
    doc.SetTitle("My First PDF")
    doc.SetFont("helvetica", "", 12)

    page := doc.AddPage(folio.A4)
    page.TextAt(20, 20, "Hello, World!")

    doc.Save("output.pdf")
}

Coordinates use top-left origin with millimeters as the default unit. A4 is 210mm × 297mm.

// Default unit is millimeters. Other units available:
// state.UnitPt (points), state.UnitCM, state.UnitInch
doc := folio.New(folio.WithUnit(folio.UnitPt))    // points (1/72 inch)
doc := folio.New(folio.WithUnit(folio.UnitInch))  // inches
doc := folio.New(folio.WithUnit(folio.UnitCM))    // centimeters

Text Drawing

TextAt — absolute positioning

page.TextAt(20, 30, "Placed at exact coordinates")

Cell — single-line box

// Cell(width, height, text, border, align, fill, ln)
page.Cell(80, 8, "Left",   "1", "L", false, 0)   // border all sides, left align

ParameterValues

align"L" left, "C" center, "R" right
ln0 cursor right, 1 next line, 2 below

MultiCell — word-wrapped

Images

f, _ := os.Open("photo.jpg")
doc.RegisterImage("photo", f)
f.Close()

page.DrawImageRect("photo", 20, 20, 60, 40)  // x, y, w, h

Images are deduplicated by content hash — register the same image twice and it appears only once in the PDF.

Complex Tables

Use AddRow/AddHeader + Render() for colspan, rowspan, multi-line wrapping, and per-cell styling.

tbl := folio.NewTable(doc, page)
tbl.SetWidths(40, 40, 40, 40)
tbl.SetCellPadding(2)

// ColSpan: title spanning all 4 columns
tbl.AddHeader(folio.TableCell{Text: "Quarterly Report", ColSpan: 4, Align: "C"})

// Regular header row
tbl.AddHeader(
    folio.TableCell{Text: "Region"},
    folio.TableCell{Text: "Q1"},
    folio.TableCell{Text: "Q2"},
    folio.TableCell{Text: "Total"},
)

// RowSpan: "North" spans 2 rows
tbl.AddRow(
    folio.TableCell{Text: "North", RowSpan: 2},
    folio.TableCell{Text: "100"},
    folio.TableCell{Text: "150"},
    folio.TableCell{Text: "250"},
)
tbl.AddRow(
    // column 0 occupied by rowspan
    folio.TableCell{Text: "200"},
    folio.TableCell{Text: "180"},
    folio.TableCell{Text: "380"},
)

// Per-cell styling
red := &folio.CellStyle{TextColor: [3]int{255, 0, 0}}
tbl.AddRow(
    folio.TableCell{Text: "Total", ColSpan: 3, Align: "R"},
    folio.TableCell{Text: "630", Style: red},
)

tbl.Render()

// Keep a block on one page

    page.Cell(0, 8, "Title", "", "L", false, 1)
    page.MultiCell(0, 5, longText, "", "L", false)
})

// Stack blocks vertically with spacing

Compress PDF

// Basic: FlateDecode + object deduplication

// Re-encode JPEG images at lower quality
folio.CompressPDF("photos.pdf", "smaller.pdf",
    folio.CompressImageQuality(60),
)

CompressImageQuality(q)0 (off)JPEG quality 1-100

Decrypt PDF

// Remove password protection
folio.DecryptPDF("locked.pdf", "unlocked.pdf", "mypassword")

// Works with user or owner password
folio.DecryptPDF("locked.pdf", "unlocked.pdf", "ownerpass")

// Unencrypted PDFs are copied as-is
folio.DecryptPDF("plain.pdf", "output.pdf", "")

Supports PDF Standard Security Handler V=1, R=2 (40-bit RC4).

PDF to Image

// Convert all pages to PNG
paths, err := folio.ConvertToImages("doc.pdf", "images/")

// Specific pages as JPEG at 300 DPI
paths, err := folio.ConvertToImages("doc.pdf", "images/",
    folio.WithFormat(folio.JPEG),
    folio.WithDPI(300),
    folio.WithPages(1, 3, 5),
)

Requires pdftoppm, mutool, or gs on PATH.

Architecture

Folio uses a 4-layer architecture with clear separation of concerns:

LayerPackageResponsibility
Public APIfolioUser-facing types and functions
Stateinternal/state, internal/resourcesDocument state, font/image registries
Contentinternal/contentPDF content stream operations
Coreinternal/pdfcorePDF serialization, reader/parser

All PDF tools (split, merge, watermark, compress, decrypt) are implemented in pure Go using the pdfcore reader. Zero external dependencies.
